🔄 入門概念

什麼是樞紐分析表？

Pivot Table 是 Excel 最強大的資料分析工具，不需要寫公式，用拖曳就能在幾秒內將幾千筆原始資料變成清晰的彙總報表。

0
需要寫的公式
3秒
建立完整報表
∞
資料視角切換
百萬+
可處理資料筆數

核心功能一覽

📊
自動彙總
數千筆交易明細，一鍵彙總為部門/月份/產品的加總、平均、計數報表。
SUM / COUNT / AVERAGE
🔄
靈活旋轉
同一份資料，隨時切換「以部門看月份」或「以月份看部門」，秒速重組報表結構。
Pivot = 旋轉視角
🎛️
互動篩選
搭配交叉分析篩選器（Slicer），點一個按鈕即可切換地區、季度、產品類別。
Slicer 篩選器
📈
即時更新
原始資料修改後，只需按「重新整理」，所有報表和圖表自動同步，不需重做。
Refresh 自動更新

📋 資料準備

資料準備原則

樞紐分析的品質取決於原始資料的品質。遵守這五個原則，讓資料隨時可分析。

1
每欄必須有標題
第一列必須是欄位標題，每個欄位都要有名字。空白標題或合併儲存格的標題會讓樞紐分析無法正確建立。
✓ 日期｜業務員｜地區｜產品｜金額
✗ A欄空白 或 B欄「業務/地區」（合併）
2
每列是一筆完整記錄，禁止合併儲存格
資料區域不能有合併儲存格。每一列都是獨立記錄，欄位值不能跨列合併。
✗ 「北區」合併了 B2:B4 三格
✓ 每列都完整填入「北區」
3
資料中不能有空白列
中間有空白列會讓 Excel 誤以為資料到此結束，只讀取空白列以前的部分。
✗ 第 5 列是空白列 → Excel 只讀到第 4 列
✓ 500 筆資料連續排列，無空行
4
同欄資料型態一致
「金額」欄不能混有文字（如「38000元」），「日期」欄要是真正的日期格式，不能是純文字字串。
✗ 金額欄混用：38000、「NT$45,000」、42000元
✓ 金額欄純數字：38000、45000、42000
5
建議先轉成「表格」（Ctrl+T）
將資料轉成 Excel 表格後，新增資料時樞紐分析的範圍會自動擴展，只需重新整理即可同步。
插入 → 表格 → 勾選「我的表格有標題」
之後樞紐分析來源選這個表格名稱
⚠️ 開始建立樞紐分析前，先按 Ctrl+End 確認資料範圍是否正確，避免遺漏資料或包含多餘空白。
🚀 建立步驟

建立樞紐分析表 — 完整步驟

從空白工作表到完整報表，跟著六個步驟，第一次就成功。

1
點選資料範圍中的任一儲存格
只要游標在資料範圍內，Excel 會自動偵測整個資料表，不需要先選取全部資料。
→ 點選 A1（或資料區域中任何有內容的儲存格）
2
插入 → 樞紐分析表
在功能區點選「插入」分頁，找到最左側的「樞紐分析表」按鈕並點擊。
功能區：插入 ▶ 樞紐分析表 ▶ 從表格/範圍
3
確認資料範圍與放置位置
確認來源範圍正確（通常 Excel 會自動偵測），選擇放到「新工作表」（建議），或現有工作表的指定位置。
來源：工作表1!$A$1:$E$501 　位置：新工作表（推薦）
4
拖曳欄位到四個區域
右側出現「樞紐分析表欄位」面板。將欄位拖曳到列、欄、值、篩選四個區域，報表即時更新。
例：地區 → 列 　產品 → 欄 　金額 → 值 　季度 → 篩選
5
調整彙總方式（如需要）
「值」區域預設是加總（SUM）。右鍵點選數值欄位 → 值欄位設定，可改為計數、平均、百分比等。
值欄位設定：加總 / 計數 / 平均值 / 最大值 / 最小值 / %
6
美化格式（選擇性）
在「樞紐分析表設計」分頁選擇報表版面、套用樣式、顯示或隱藏總計列，讓報表更易閱讀。
設計 → 樞紐分析表樣式 → 選擇喜歡的配色主題
💡 資料更新後：點選樞紐分析表任一儲存格 → 右鍵 → 重新整理（或快捷鍵 Alt+F5），報表即時同步。
🗂️ 四大區域

四大區域解析

樞紐分析表的核心是四個欄位區域。理解每個區域的作用，就能自由組合出任何報表。

📋
列（Rows）
決定報表「橫向分類」。放在這裡的欄位成為報表左側的列標籤，每個唯一值獨佔一列。
常放：地區、部門、業務員、產品類別
列標籤
📐
欄（Columns）
決定報表「縱向分類」。放在這裡的欄位成為報表頂部的欄標籤，交叉產生矩陣式報表。
常放：季度、年份（類別少的欄位）
欄標籤
🔢
值（Values）
要計算的數值欄位。支援加總、計數、平均、最大值、最小值、百分比等多種彙總方式。
常放：金額、數量、利潤等數值欄位
彙總數值
🔍
篩選（Filters）
報表頂部的全局篩選器。選擇特定值後，整個樞紐分析表都會篩選到只顯示該條件的資料。
常放：年度、負責人等全局條件欄位

🧩 互動建立器

樞紐分析互動建立器

拖曳欄位到四個區域，即時預覽樞紐分析結果。體驗真實的操作邏輯。

彙總方式：

可用欄位

📋 列（Rows）

🔢 彙總方式

彙總方式大全

值區域不只有「加總」，根據分析目的選擇合適的彙總方式，能揭露不同的洞察。

彙總方式用途說明適用場景設定路徑
加總 SUM所有數值的總和銷售總額、總費用預設
計數 COUNT有值的儲存格數量訂單筆數、員工人數值欄位設定
平均值 AVG所有數值的平均平均客單價、平均薪資值欄位設定
最大值 MAX範圍內最大的數值最高業績、最高分數值欄位設定
最小值 MIN範圍內最小的數值最低售價、最短工時值欄位設定
佔比 %各項目佔總計的百分比各地區佔總銷售比顯示為→總計%
差異 DIFF與參照項目的差異vs 上月、vs 上年同期顯示為→差異
累積 RUNNING累積加總 Running Total年度累積業績顯示為→累積加總

交叉分析篩選器（Slicer）

Slicer 用按鈕取代下拉篩選，讓報表變成可互動的儀表板，是製作 Excel Dashboard 的關鍵功能。

互動示範 — 點選篩選條件

產品

新增 Slicer 步驟

1
點選樞紐分析表任一儲存格
必須先選到樞紐分析表，才能插入屬於它的 Slicer。
2
樞紐分析表分析 → 插入交叉分析篩選器
勾選要做成按鈕的欄位（可多選），確定後出現浮動的 Slicer 元件。
3
排列位置，調整欄數與樣式
拖曳 Slicer 到合適位置，調整欄數讓按鈕橫排，在格式分頁自訂顏色。
💡 一個 Slicer 可以同時控制多個樞紐分析表！右鍵 Slicer → 報表連結，完成儀表板互動效果。
⚙️ 計算欄位

計算欄位與計算項目

樞紐分析表內建「計算欄位」，可在裡面寫公式，讓彙總後的資料再做一層運算。

計算欄位（Calculated Field）

ℹ️ 計算欄位是在原始欄位之間建立新的計算關係，例如用「金額」÷「數量」算出「平均單價」，不需修改原始資料。

常見計算欄位範例

// 利潤率
= 利潤 / 銷售額

// 平均客單價
= 金額 / 訂單數

// 稅後金額（含稅5%）
= 金額 * 1.05

新增步驟

1
點選樞紐分析表任一儲存格
2
樞紐分析表分析 → 欄位、項目及集合 → 計算欄位
3
輸入名稱（如「利潤率」），公式欄輸入 = 利潤/銷售額
4
按「新增」→「確定」，新欄位出現在欄位清單中

計算項目（Calculated Item）

ℹ️ 計算項目是對列/欄標籤的項目做計算，例如將「北區」+「中區」合併成「北中合計」新項目。
地區銷售額說明
北區671,000原始項目
中區567,000原始項目
南區496,000原始項目
北中合計1,238,000計算項目：= 北區 + 中區
⚠️ 計算項目可能與總計列發生重複計算。建議關閉總計列：設計 → 總計 → 關閉列和欄。

⚖️ 比較

樞紐分析 vs 一般函式

樞紐分析和 SUMIFS/COUNTIFS 各有優勢，了解差異選擇最適合的工具。

🔄 樞紐分析表 推薦探索用
✓不需公式，拖曳即可
✓秒速切換報表結構
✓自動處理所有分類
✓搭配 Slicer 做儀表板
✓資料更新只需重新整理
✗結果不能被其他公式引用
✗格式難以完全自訂
✗版面位置相對固定
📐 SUMIFS / COUNTIFS 推薦固定報表
✓結果可被其他公式引用
✓版面完全自由設計
✓可整合到既有報告模板
✓支援複雜 AND/OR 條件
✗每格都要手動寫公式
✗新增分類時公式要跟著改
✗無法快速切換分析角度
✗大量條件下公式很複雜

相同任務，兩種做法

任務：算出北區筆電 Q1 的銷售總額

方法 A — SUMIFS 公式（固定一格）：
=SUMIFS(金額, 地區, "北區", 產品, "筆電", 季度, "Q1")
方法 B — 樞紐分析（自動產生所有組合）：
列：地區（北/中/南）　欄：產品（筆電/手機/平板）　篩選：季度=Q1
→ 自動產生 3×3 = 9格完整交叉報表，無需寫任何公式
ℹ️ 建議原則：資料探索、主管簡報、一次性分析 → 用樞紐分析。固定格式報告、需要引用結果、複雜邏輯 → 用函式。
